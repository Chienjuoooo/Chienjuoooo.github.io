  • 學程宗旨與特色
    • 教育目標

      1. 醫學資訊專業知識及問題解決。
      2. 獨立思考及創新。
      3. 溝通、協調及合作。
      4. 跨領域人文關懷精神。
      5. 自主學習。


    • 核心能力

      1. 資訊系統分析、設計、及實作的能力。
      2. 運用科技及創新的能力。
      3. 溝通協調及團隊合作能力。
      4. 口語表達及文件撰寫能力。
      5. 跨領域人文關懷能力。
      6. 自我反思及自主學習的能力。


  • 未來出路

    1. 升學：國內外資訊或醫學資訊相關研究所。
    2. 就業：一般公務員、資訊技師、基因分析師、藥物設計師、醫院資訊系統開發、醫療系統開發、醫療儀器及系統銷售、醫療雲端系統開發、醫療儀器開發、資訊系統開發、資訊系統軟硬體銷售、專業研究人員、中或小學教師、自行創業
